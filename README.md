Static site generator.


Overview
--------

Aldous is a bare bones static site generator that doesnâ€™t rely on file I/O. It is designed to be used as a backend for CLI and/or GUI applications. Its architecture is highly inspired by [Metalsmith](http://www.metalsmith.io/).
